# Identidad de marca — Evalúa Fácil

Assets oficiales del logotipo. Todos con los colores originales de la marca
(carpeta azul, cheque teal, texto azul marino / "Fácil" azul). No existe una
variante de texto blanco para fondos oscuros — **el logo siempre se coloca
sobre un fondo blanco**, tanto en la web como en cualquier material.

## Archivos

| Archivo | Qué es | Cuándo usarlo |
|---|---|---|
| `logo-icono-transparente.png` | Solo el icono (la hojita), fondo transparente | Sobre un fondo ya blanco/claro; favicons, esquinas compactas |
| `logo-icono-fondo-blanco.png` | Solo el icono, con fondo blanco horneado | Cuando necesitas un PNG que ya trae su propio fondo blanco |
| `logo-completo-transparente.png` | Logo completo (icono + "Evalúa Fácil" + subtítulo), fondo transparente | Sobre fondo blanco/claro; es el que usa la app en Landing/Login |
| `logo-completo-fondo-blanco.png` | Logo completo con fondo blanco horneado | Material que no garantiza un fondo claro detrás |

## Regla de uso

- **Siempre sobre blanco.** Las versiones transparentes son para colocar sobre
  superficies que ya son blancas o muy claras. Si el fondo es de color (p. ej.
  el azul del panel lateral dentro de la app), se envuelve el logo en un
  contenedor blanco — nunca se usa una versión de texto blanco.
- Las fuentes en la app viven en `public/logo-evalua-facil.png` (completo) y
  `public/logo-icon.png` (icono), ambas transparentes. Esta carpeta es la
  referencia de marca / distribución; los assets aquí se derivan de esas fuentes.
